O Disparador de mensagem é um projeto Iniciante/Basico.

Nele foi utilizado o html, e css para marcação e estilo da pagina.
O javascript foi implantado com um metodo "function" para abrir um "Log" com um "prompt" para ser digitado o numero de WhatsApp, e logo apos uma função window.open para validar e abrir o WhatsApp web em uma nova guia do navegador. Caso não houver nenhum numero digitado ou aperte o botão "Cancelar" o "Log" se fecha e volta para a pagina inicial.


Modo de usar:

- Clique na mensagem escolhida.
- Digite o numero no input que irá aparecer na parte superior da tela
- Apos isso, clique em "OK" .
- Você vai ser redirecionado para o WhatsApp Web.
- Agora é so confirmar e enviar a mensagem.
